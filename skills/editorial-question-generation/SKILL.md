# editorial-question-generation

## Description
Generate questions for Anthony to answer during editorial sessions.

## When to use
At the start of an editorial session, before drafting.

## Rules
1. Read voice memory first. Don't ask about positions already established.
2. Read last 7 days of ingestion. Questions should reference current events.
3. Evaluate 3-5 candidates. Pick the most timely and useful.
4. Confirmation questions first ("do you still hold X position on Y?"), then disambiguation ("your view on A seems to conflict with B, which takes priority?"), then open questions last.
5. Max 6 questions per session. Order by speed to answer (yes/no first, open-ended last).
6. Include links to relevant ingestion items so Anthony has context.
7. Frame questions directly: "YouTube lowered its threshold. Does this change your position on micro-creators being commercially viable below 1k subs?"
