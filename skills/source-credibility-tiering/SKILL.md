# source-credibility-tiering

## Description
Assess source credibility when curating newsletter items.

## When to use
During curation, when deciding which source to cite for a story.

## Rules
Five tiers:
- 1.0: Official sources, primary documents (YouTube blog, Twitch announcements, SEC filings, published research)
- 0.8: Established journalism (The Verge, TechCrunch, Ars Technica, MIT Tech Review, Financial Times)
- 0.6: Aggregators and curated newsletters (Ben's Bites, Latent Space, Every, Tubefilter)
- 0.4: Creator commentary (individual creator blogs, podcast transcripts, YouTube essays)
- 0.2: Social and community (Reddit, Hacker News, Twitter/X threads)

Never cite:
- Paywalled content the reader can't access
- Known misinformation sources
- AI-generated content farms
- Press releases presented as journalism

When two sources cover the same story, cite the higher-tier source.
